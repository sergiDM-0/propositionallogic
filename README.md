Este proyecto es una evolución de un código base anterior, con una **reescritura significativa** (90%+) para mejorar el rendimiento y la estabilidad.

Los cambios principales incluyen:
* **Corrección del Parser Lógico:** Solución al error de recursividad en operadores unarios (como la negación `~`).
* **Arquitectura Modular:** Separación clara entre la lógica de evaluación (`truth_table`) y la interfaz de usuario (`cli`).
* **Manejo de Errores:** Implementación de validaciones para evitar cierres inesperados (`KeyError`).
* **Licenciamiento:** Estandarización bajo **GNU AGPLv3** para garantizar el software libre.
